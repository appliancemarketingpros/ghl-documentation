# **What is Editing Default Email Templates in Client Portal?**

  


The Edit Default Email Templates capability lets Client Portal Owners generate an editable copy of any system default email used by portal apps (e.g., Documents & Contracts, Courses, Certificates). Instead of building from scratch, you convert the default, customize the content, and set it as the active template for that notification.

* * *

## **Key Benefits of Edit Default Email Templates**

  


  * **Speed:** Instantly convert a default into an editable template—no need to recreate from scratch.  
  


  * **Consistency:** Keep the original layout while refining tone and content for your audience.  
  


  * **Personalization:** Use merge fields/custom values to add dynamic details.  
  


  * **Accuracy:** Preview with custom fields so you know exactly what contacts will see.  
  


  * **Control:** Apply per‑notification edits across supported Client Portal apps.

Following a clear setup sequence reduces errors and ensures your customized template activates for the intended notification.

  


  1. Navigate to **Sites → Client Portal → Settings → Email Settings**  
  
![](https://jumpshare.com/share/IaruuKABWh6R9nVuUjnX+/Screenshot+2025-12-05+at+6.22.02%E2%80%AFPM.png)  
  


  2. **or Memberships → Client Portal → Settings → Email Settings.**  
  
**![](https://jumpshare.com/share/fQMnDfMSVjHNux1iX93F+/Screenshot+2025-12-05+at+6.29.05%E2%80%AFPM.png)**  
  


  3. Locate the**Main Notification Features (Client Portal, Communities, Certificates, Courses).**  
  
**![](https://jumpshare.com/share/pRUHX38LQI3CGS1kS4KC+/Screen+Shot+2025-12-05+at+6.40.49+PM.png)**  
  


  4. Under the main features, for each sub-category notification, click on the**Eye Icon** to preview the template and click on **Default Template** button to start customizing.  
  
![](https://jumpshare.com/share/kDAC8pMKRH6UG177SFP2+/GIF+Recording+2025-12-05+at+6.42.53+PM.gif)  
  


  5. You can click on the **Select** **Template** dropdown to select a previously saved email template as well.

## **Frequently Asked Questions**

  


**Q. Who can edit default email templates in the Client Portal?**  
Client Portal Owners with access to **Email Settings**. Role‑based visibility may vary by account configuration.   


**Q. Does saving a custom copy immediately activate it?**  
Yes. After saving, the customized version is used for that specific notification.   


  


**Q. Can I manage templates per function (e.g., Courses, Certificates, Documents & Contracts)?**  
Yes. Each supported function lists its notifications in **Email Settings** with template controls.  


  


**Q. Can I select templates I created previously?**  
They remain available. You can replace a notification’s default with your previously saved Email template.  
  
**Q: Can I switch back to the default template?**

Yes.
