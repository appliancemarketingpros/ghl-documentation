Address Auto-Complete is now **free and unlimited** in all HighLevel accounts. This feature makes it easier for your leads and customers to enter accurate address details in forms and surveys. As users type into the Address field, real-time suggestions appear and automatically fill in the Street, City, State, Country, and Postal Code fields — ensuring fast, accurate, and consistent data collection.

* * *

## **What is Address Auto-Complete?**

  


Address Auto-Complete converts the standard Address field in HighLevel forms and surveys into a smart, predictive search bar. It uses an integrated global database to surface relevant address suggestions as users type, making form submission faster and error-free.

  


Accurate address data is essential for workflows, segmentation, reporting, and delivery-based actions. Address Auto-Complete eliminates typing errors, guarantees consistent formatting, and speeds up form completion, improving both user experience and data reliability.

  


### **Smart Search Suggestions**

  


As a user types into the Address field, HighLevel provides live predictive suggestions based on real-world address data.

  


### **Auto-Fill Address Fields**

  


When the user selects an address from the dropdown, all relevant fields (Street, City, State, Country, and Postal Code) are automatically filled. This ensures consistent formatting and eliminates manual data entry errors.

## **How to Enable Address Auto-Complete**

  


  


### **Add the Address Field**

  


Navigate to **Sites → Forms** (or **Surveys**) and open the form builder where you want to use Address Auto-Complete.

  


In the **Form Elements** panel, drag the **Address** field into your form layout. The Address element is marked as **Updated** , indicating that it supports Auto-Complete functionality.

  


![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758329/original/1keSyOM9IjxVc9cggUh6vpY5sReynZIdUQ.png?1760147263)

  


###   


### **Access Address Field Settings**

  


After adding the Address field, click on it to open the settings panel on the right side of the screen.

  


This panel allows you to customize address behavior, enable auto-complete, and control whether users must select a verified address.

  


![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758336/original/pFSPtlYwSv71eVBrmuOAC8FmwNJFsykfug.png?1760147347)

###   


  


### **Enable Auto-Complete Address**

  


In the settings panel, locate the toggle labeled **Auto-Complete Address** and switch it **ON**.

  


Once activated, this setting transforms the standard address input into a smart search bar that suggests accurate addresses as users type.

  


  


![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758423/original/LW7dk4ZxDjLJEfzjjsy-vKEb3AyAzSizFQ.png?1760147656)

  


###   


### **Make Address Selection Mandatory **

  


If you want to ensure that users only submit validated addresses, toggle **Mandatory to Select Address in Search Bar** to **ON**.

When enabled, users must pick one of the suggested addresses before they can submit the form. If left OFF, they can enter an address manually.

  


###   


![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758426/original/RxZTM2OVNrnwIzCdd8ip3sbi6f63t_is_w.jpeg?1760147670)

  


  


### **Preview the Form**

  


Click **Preview** at the top right of the form builder to see how the Address field appears in the live form.

The Address field will now display a **search bar** , allowing users to begin typing and instantly view address suggestions as they type.

  


![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758436/original/zGD2LeIC6RMs_jDGsds9JgI1fDnGEbLNOQ.jpeg?1760147782)

###   


### **Test Live Address Suggestions**

  


In the preview, type a few characters (such as part of a street name) into the Address field.

You’ll see predictive suggestions appear below the search bar — selecting one will automatically populate the Street, City, State, Country, and Postal Code fields.

  


![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155055758434/original/_pK5PJkq8QF9qLUAgJBfLtgDmsd7GQeIrA.jpeg?1760147771)

###   


### **Save and Publish**

  


After confirming that Auto-Complete works as expected, click **Save** to apply your changes.

Once published, every visitor who fills out the form will experience fast, accurate address entry powered by built-in Auto-Complete.

## **Frequently Asked Questions**

  


**Q: Is Address Auto-Complete still a paid feature?**

No. It’s now completely free and unlimited for all HighLevel accounts.

  


**Q: Can I still allow users to manually type their address?**

Yes. You can disable the Auto-Complete toggle if manual entry is preferred.

  


**Q: Does Address Auto-Complete work globally?**

Yes. It supports most countries and postal formats worldwide.

  


**Q: Will my old forms automatically get Auto-Complete enabled?**

No. You’ll need to open each form or survey and enable the toggle manually.

  


**Q: What happens if a user doesn’t select a suggested address?**

If the “Mandatory to Select Address in Search Bar” toggle is ON, users must pick a suggestion to submit the form. If OFF, they can type freely.

  


**Q: Does Auto-Complete overwrite existing contact data?**

No. Submissions follow your usual contact update rules and mapping preferences.
