# Why does my Domain not show the Edit Button?

**Source URL:** [https://help.gohighlevel.com/support/solutions/articles/48001161158-why-does-my-domain-not-show-the-edit-button-](https://help.gohighlevel.com/support/solutions/articles/48001161158-why-does-my-domain-not-show-the-edit-button-)  
**Category:** Sites  
**Folder:** Troubleshooting Funnels

---

Edit button is only shown on one location, where you first add the domain. It will not be shown in other locations since the default page can only have one value.
