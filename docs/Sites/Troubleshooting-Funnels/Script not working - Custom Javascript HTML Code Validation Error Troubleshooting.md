# Script not working? - Custom Javascript / HTML Code Validation Error Troubleshooting

**Source URL:** [https://help.gohighlevel.com/support/solutions/articles/48001159729-script-not-working-custom-javascript-html-code-validation-error-troubleshooting](https://help.gohighlevel.com/support/solutions/articles/48001159729-script-not-working-custom-javascript-html-code-validation-error-troubleshooting)  
**Category:** Sites  
**Folder:** Troubleshooting Funnels

---

If your custom code has the <script> tag within a <div> tag, please cut the script tag and paste it below the <div> tag so that both tags are separate. The <script> tag does not work within a <div> tag as per our funnel/website builder settings.
