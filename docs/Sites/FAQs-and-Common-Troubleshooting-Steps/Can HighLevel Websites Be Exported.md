# Can HighLevel Websites Be Exported?

**Source URL:** [https://help.gohighlevel.com/support/solutions/articles/155000007342-can-highlevel-websites-be-exported-](https://help.gohighlevel.com/support/solutions/articles/155000007342-can-highlevel-websites-be-exported-)  
**Category:** Sites  
**Folder:** FAQs and Common Troubleshooting Steps

---

HighLevel websites are built using a proprietary page builder and **do not have a supported method for exporting a website for use on another platform**.

This article explains what is and is not supported when customers attempt to export HighLevel websites.

* * *

## **Can HighLevel Websites Be Exported?**

**No.**  
HighLevel does not support exporting websites in a way that allows them to be managed or edited outside the HighLevel platform.

There is no supported 1:1 website export.

* * *

## **About Copying Website HTML**

From a technical standpoint, the rendered HTML output of a website can be copied and hosted elsewhere.

When this is done with a HighLevel website:

  * The copied page is no longer managed by HighLevel

  * The page is not editable through the HighLevel builder

  * HighLevel does not provide support for externally hosted copies

  * Continued functionality depends on the original HighLevel location and assets remaining active


This is **not considered a supported website export**.

* * *

## **How to Copy Page HTML (Unsupported)**

If you choose to copy the rendered HTML of a HighLevel page, this can be done directly from a web browser.

**Example method:**

  1. Open the published page in your browser

  2. Right-click anywhere on the page and select **View Page Source** (or similar)

  3. Copy the HTML content and save it as an `.html` file


This method copies the **rendered output of the page only**.

> HighLevel does not provide tools, guidance, or support for copying, hosting, or maintaining externally hosted HTML files.

* * *

## **What Copied HTML May Be Used For**

Copied HTML may be used for:

  * Referencing or recovering content

  * Manually rebuilding the website on another platform

  * Archival or record-keeping purposes
