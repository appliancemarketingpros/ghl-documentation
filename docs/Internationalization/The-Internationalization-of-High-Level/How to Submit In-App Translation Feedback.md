# **What is the Translation Feedback Feature?**

  


Report Translation Feedback is a built-in feature that surfaces a **report icon in the top bar** so eligible users can submit translation issues without leaving the current page. Submissions help the product team quickly identify strings that need review, improving accuracy and consistency across the platform.  
  

    
    
    **Note:** The feature is **not visible** when the platform**language** is**** set to**English**.  
    

  


  * Submit feedback **from the page where you see the issue.**  
  


  * Attach a **screenshot** to highlight the exact string.  
  


  * Your report is routed for review; you’ll be **notified when it’s fixed.**


* * *

## **Key Benefits of Translation Feedback Feature**

  


Understanding the benefits helps you decide when to use the in-app report versus opening a general support ticket.

  


  * **Real-time capture** : Report issues in the moment, from the exact screen you’re using.  
  


  * **Lower effort** : Avoid external tickets for routine translation fixes.  
  


  * **Higher quality** : Add screenshots so reviewers can pinpoint the exact string.  
  


  * **Faster iteration** : Centralized reports streamline localization improvements.


* * *

## **How to Use the Translation Feedback Feature**

  


Follow these steps to submit actionable translation feedback directly from the UI.

  


  1. Confirm you’re logged in as **Agency Owner/Admin** and your language is **not English**.  
  

  2. Open the page where the translation issue appears.  
  

  3. Click the **Report Translation Issue** icon (top bar).  
  
![](https://jumpshare.com/share/HnGm7ViM2dA1ylQO2LjV+/Screen+Shot+2025-09-15+at+8.11.17+PM.png)  
  

  4. The content of the panel will appear in the selected platform language.  
  

  5. Attach a **screenshot** (mandatory) that clearly shows the problematic area and its location.  
  
![](https://jumpshare.com/share/dz0ne7Of1YemNPH5Emvj+/Screen+Shot+2025-09-15+at+8.15.27+PM.png)  
  

  6. Select the **issue type** (e.g., incorrect translation, grammar, incorrect formatting).  
  
![](https://jumpshare.com/share/3B6FCm7pj5mk16opqtZu+/Screen+Shot+2025-09-15+at+8.18.33+PM.png)  
  

  7. Add a **brief description**.  
  
![](https://jumpshare.com/share/VEssGpFoVnqvkWLq2UXj+/Screen+Shot+2025-09-15+at+8.20.26+PM.png)  
  

  8. Under **Where Did You See This?** section, add the **path, e.g. Dashboard >Funnel>Step Builder**.**  
**  
![](https://jumpshare.com/share/ko4QrRgXcpgczqq4QkGa+/Screen+Shot+2025-09-15+at+8.23.40+PM.png)  
  

  9. Click the**Report Issue** button.   
  

  10. You’ll be informed when the issue is resolved.  
  
![](https://jumpshare.com/share/vUne13Vnw47OexaRpfCg+/Screen+Shot+2025-09-15+at+8.24.54+PM.png)

## **Frequently Asked Questions**

**  
Q: Who can access the Report Translation Feedback feature?  
** Agency Owner and Agency Admin roles.

**  
**

**Q: I can’t see the icon, why?**

The feature is hidden for English language settings and only appears for Agency Owner/Admin roles.

**  
**

**Q: What should I include in my report?**

A short description, the correct wording, and a screenshot showing the issue in context.

**  
**

**Q: How will I know when it’s fixed?**

You’ll be informed when the reported translation is resolved.
