## **Frequently Asked Questions**

  


**Q: Is the desktop app different from the HighLevel web app?**  
The experience and data are the same; the desktop app packages HighLevel in a dedicated, optimized window with native notifications.

  


**Q: Can I use multiple accounts or sub‑accounts?**  
Yes - sign in normally, then switch using the account menu if you have access to multiple sub‑accounts.

  


**Q: Will my whitelabeled domain affect login?**  
Yes - if enabled, you’ll be routed through your branded login during sign‑in before returning to the app.

  


**Q: What if I don’t see notifications?**  
Enable OS‑level notifications for HighLevel and then review in‑app preferences.
