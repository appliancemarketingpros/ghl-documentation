**Mobile UI Improvements for E-commerce Stores**

HighLevel has introduced a set of mobile-focused UI improvements across E-commerce store pages. These updates enhance layout consistency, usability, and responsiveness to create a smoother shopping experience on mobile devices.

* * *

## **Overview**

These improvements apply automatically to E-commerce stores and focus on optimizing key customer touchpoints, including product browsing, reviews, cart, checkout, and post-purchase pages.

No setup or configuration is required.

* * *

## **Product & Browsing Improvements**

### **Product Ratings Display**

Product ratings are now simplified on mobile. A single star icon with the rating count appears on product images, reducing visual clutter while maintaining clarity.

### **Improved Sort & Filter Experience**

The sort and filter interface has been redesigned for mobile:

  * Cleaner and more intuitive layout
  * Price filter now fits properly on smaller screens
  * Eliminates previous overflow issues


* * *

## **Reviews Experience Improvements**

### **Write a Review Modal**

The review submission experience has been updated for better clarity and usability on mobile devices.

### **Dropdown Display Fix**

Dropdown overflow issues in the “Create Review” section have been resolved, ensuring all options are visible and accessible.

* * *

## **Cart & Checkout Improvements**

### **Cart Drawer and Cart Page**

Cart layouts have been refined to improve visibility of:

  * Product details
  * Subscription information
  * Error messages


### **Checkout Page Enhancements**

  * Accordion-style cart summary allows users to expand or collapse order details
  * Coupon field has been repositioned for better visibility alongside subtotal, shipping, taxes, and total breakdown


* * *

## **Post-Purchase Improvements**

### **Thank You Page Update**

The thank you page layout has been reorganized to improve clarity and make important order details easier to find after purchase.

* * *

## **Important Notes**

  * These updates apply automatically to all E-commerce stores
  * Changes are specific to mobile devices
  * No action is required from users
