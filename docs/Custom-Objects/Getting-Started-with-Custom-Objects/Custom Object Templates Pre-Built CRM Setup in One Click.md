# **What is Custom Object Templates?**

  


Custom Object Templates are pre-built configurations that help you quickly create Custom Objects for common industries and use cases. Instead of manually deciding which objects, fields, folders, and associations to create, you can start with a ready-made structure and adjust it to fit your business needs.

  


Custom Objects are used to track business data that does not fit neatly into standard CRM records like Contacts, Companies, or Opportunities. Templates make that setup faster by giving you a practical starting point based on common industry workflows.

  


Custom Object Templates are available in the Template Library alongside other template modules in HighLevel. Users can browse supported industries, preview template details, and apply a template when creating Custom Objects.

* * *

## **Key Benefits of Custom Object Templates**

  


Custom Object Templates help teams launch structured CRM setups faster while still allowing flexibility for industry-specific processes.  
  


  * **Faster Setup:** Create Custom Objects, fields, folders, and associations in one flow instead of building each part manually.  
  

  * **Industry Specific Starting Points:** Choose from templates designed for common industries such as Real Estate, Insurance, Automotive, SaaS, Events, Pets, and Home Services.  
  

  * **Easier Data Organization:** Use predefined fields and folders to keep related information grouped clearly.  
  

  * **Built In Relationship Mapping:** Templates can include associations so records can be connected across Companies, Opportunities, and Custom Objects.  
  

  * **Preview Before Applying** : Review the template’s objects, fields, associations, description, tags, and prerequisites before using it.  
  

  * **Agency Visibility Control:** Agency Admins can manage which template modules or categories are visible to sub-accounts through Template Library visibility settings.


* * *

## **Agency Visibility Controls for Custom Object Templates**

  


Agency visibility controls help agencies keep the Template Library focused for each sub-account. This is useful when an agency serves a specific niche and only wants clients or team members to see relevant template categories.

  


Agency Admins can manage Custom Object Template visibility using the existing Template Library visibility controls. 

  


These controls can be used to show or hide template access for specific sub-accounts or product areas. Hiding Template Library access restricts browsing or importing new templates, while assets already created from templates remain available for editing.

* * *

## **How To Setup Custom Object Templates**

  


Applying a Custom Object Template gives your account a ready-made object structure that can be used as a starting point for your CRM setup. Reviewing the template before applying it helps confirm the objects, fields, and associations match your business process.

  


To use a Custom Object Template:  
  


  1. Go to Settings in the sub-account.  
  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155070636537/original/sXcWd7eHIOiJoy8_znMI3EwAzEd2T5dmng.png?1777997853)  

  2. Select Objects in the left pane.  
  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155070636820/original/cJV3hneMHx-DSsJ8M7VFbvEIqaTW3l__lw.png?1777997897)  

  3. Click Add Custom Object to create a new custom object.  
  
  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155070636918/original/kmn_GITsAuxr05FWMymgktdzZXtkblrZvw.png?1777997966)  

  4. Choose Use a template to create objects using templates.  
  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155070636989/original/OXeI7mUsVwJK2eLeMtVuQRkUZLZQLHmu_w.png?1777998028)  

  5. Browse the available Custom Object Templates.  
  
  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155070637029/original/rU5U8NMIL0UICYNHFto_J7ElAcNokJBwZw.png?1777998092)  
  

  6. Select the template that matches your industry or use case.  
  

  7. Review the template preview, including objects, fields, associations, description, tags, and prerequisites.  
  

  8. Click "Import Objects" to apply or use the template.  
  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155070637228/original/jBrh_USg27JMXRjZuuyCyimWP3Wnr6F6jA.gif?1777998206)  
  

  9. Review the created Custom Objects and make any needed adjustments.


  
After the template is applied, review the created objects, fields, and associations to ensure they match your workflow. Custom Objects can be managed from the Objects area in Settings, and associations can be reviewed from the object’s association settings.

## **Frequently Asked Questions**

  


**Q: Do Custom Object Templates create records automatically?  
** A: Custom Object Templates are designed to create the CRM structure, such as Custom Objects, fields, folders, and associations. They are not described as importing customer records or sample records in the current release note.

  


**Q: Can I edit a template after applying it?  
** A: You can manage Custom Objects from the Objects area in Settings. Review the created objects, fields, and associations after applying a template and adjust the setup as needed based on your business process.

  


**Q: Will applying a Custom Object Template overwrite my existing Custom Objects?  
** A: The release note does not state that templates overwrite existing Custom Objects. Review the template preview before applying and verify the created setup afterward.

  


**Q: Can Agency Admins hide Custom Object Templates from sub-accounts?  
** A: Yes. Custom Object Templates use the existing Template Library visibility controls, allowing Agency Admins to manage access for sub-accounts.

  


**Q: What happens if a template is hidden after it has already been used?  
** A: Existing assets created from templates remain available for editing even if browsing or importing new templates is later restricted.

  


**Q: Do Custom Object Templates include associations?  
** A: Yes. Templates can include associations, helping connect related records such as Companies, Opportunities, and Custom Objects.

  


**Q: Are workflows or sample data included with Custom Object Templates?  
** A: Not in the current release. Workflows and sample data are planned for a future phase based on the release note provided.

  


**Q: Can users create and save their own Custom Object Templates?  
** A: Not in the current release. User-contributed templates are planned for a future phase based on the release note provided.

  


**Q: Which industries are currently supported?  
** A: Current templates include Real Estate, Insurance, Automotive, SaaS, Events, Pets, and Home Services.
