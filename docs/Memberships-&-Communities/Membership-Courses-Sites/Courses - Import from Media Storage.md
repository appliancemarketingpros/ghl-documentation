# **What is Import from Media Storage Feature in Courses?  
**

  


Import from Media Storage lets you attach existing audio or video files stored in HighLevel Media Storage to a course lesson. Instead of uploading the same file again, you select it from Media Storage and import it into the lesson.

  


This approach keeps your content library organized and reduces repetitive work—especially when you reuse the same video or audio across multiple lessons or courses.

* * *

## **Key Benefits of Import from Media Storage Feature in Courses**

  


  * **No duplicate uploads** : Import existing audio/video instead of uploading the same file multiple times.  
  


  * **Faster course building** : Add lesson media in a few clicks using content already in Media Storage.  
  


  * **Centralized content management** : Keep course media organized in Media Storage for easier reuse.  
  


  * **Cleaner workflows** : Reduce duplicate file versions and simplify media updates across courses.

## **How To Import Media from Media Storage into a Course Lesson**

  


Importing media directly from Media Storage helps you build lessons faster while keeping your media library centralized and reusable.

  


  1. Go to **Memberships** > **Courses > Products**.  
  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155065595345/original/8GurbWjMl4MAEI78EI3yKRscaYG1lvIM-Q.png?1771941100)  
  


  2. **Open** the **course** you want to edit by clicking on it.  
  


  3. Navigate to the lesson you want to update.  
  


  4. Click to **create** a new lesson or **edit** an existing lesson.  
  
![](https://jumpshare.com/share/6Bg2KiruBEo2dVgCeyjd+/GIF+Recording+2026-02-24+at+19.24.24.gif)  
  


  5. Click on the **Import from Media Storage** button.  
  


  6. Click on the **Click to select from media storage** button to browse your Media Storage library and select the audio/video file you want to use.  
  
![](https://jumpshare.com/share/3eYw0OSVaABgmNZBnSmA+/Screen+Shot+2026-02-24+at+19.26.00.png)

  
  


  7. **Select** the **audio/video file** and click **Insert Media**.  
  
![](https://jumpshare.com/share/4D9sfLPklAD0JIMjXMYH+/Screen+Shot+2026-02-24+at+19.29.19.png)

## **Frequently Asked Questions**

  


**Q: Are images supported?**

No, you can only import audio/video.

  


**Q: Does importing from Media Storage upload a new copy of the file?**

No. Importing uses the existing audio/video stored in Media Storage so you don’t have to re-upload.  
  


**Q: I don’t see “Import via Media Library” in my lesson. What should I check?**

Confirm you’re in the lesson editor (create/edit) and verify your user role includes access to **Medias** and **Memberships/Courses**.

  


**Q: Can I reuse the same video in multiple lessons or courses?**

Yes. Importing is designed for reuse. Select the same Media Storage file in as many lessons as needed.

  


**Q: Will learners see the imported media if the lesson isn’t published?**

Learners only see published content. Save stores your changes; publishing makes them available based on your course visibility settings.
