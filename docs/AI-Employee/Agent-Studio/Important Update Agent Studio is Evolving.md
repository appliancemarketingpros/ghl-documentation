# Important Update: Agent Studio is Evolving

**Source URL:** [https://help.gohighlevel.com/support/solutions/articles/155000007931-important-update-agent-studio-is-evolving](https://help.gohighlevel.com/support/solutions/articles/155000007931-important-update-agent-studio-is-evolving)  
**Category:** AI Employee  
**Folder:** Agent Studio

---

We're writing to let you know about an update to Agent Studio.  
As part of our ongoing commitment to delivering the best possible experience, we've introduced **SuperAgents** —our next-generation agent-building experience. Going forward, we recommend using SuperAgents for creating all new agents.

### What This Means for You

We understand that changes like this can raise questions, so here's what you can expect.

If you haven't created any agents yet: To create your first agent, enable **SuperAgents** from **Labs**. Agent Studio is no longer available for creating new agents.

If you were planning to create new agents: New agent creation is now available through **SuperAgents**. Simply enable **SuperAgents** from **Labs** to start building.

If you already have existing agents: Good news—nothing changes for you. All your existing agents will continue to work exactly as they do today. Every trigger you've set up will keep running as expected, and you'll still be able to edit, update, and manage your existing agents from your sub-account without any disruption.

### What's Next

SuperAgents is the future of agent building, and we'll continue improving it with new capabilities over time.  


  


If you have any questions or need assistance enabling SuperAgents, please don't hesitate to reach out to our support team. We're here to help.
