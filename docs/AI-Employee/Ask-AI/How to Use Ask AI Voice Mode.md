# **What is Ask AI Voice Mode?**

  


Ask AI Voice Mode is a voice-first way to interact with Ask AI in HighLevel. Instead of typing every prompt, users can speak naturally, listen to responses, interrupt or redirect the conversation, and switch back to text when needed.

  


Voice Mode builds on the existing Ask AI experience by adding spoken input and spoken responses. It can help users ask questions, retrieve information, trigger supported actions, manage workflows, and continue multi-step conversations using voice.

* * *

## **Key Benefits of Ask AI Voice Mode**

  


Voice Mode helps users work faster when typing is inconvenient or when they need to stay focused on another task. It is especially useful for multitasking, reviewing information, and moving through supported Ask AI actions conversationally.

  


  * **Hands-Free Productivity:** Speak to Ask AI naturally to ask questions, manage tasks, retrieve data, or trigger supported actions without relying only on the keyboard.


  


  * **Natural Conversation Flow:** Ask AI can respond through voice, making the experience feel more like a live conversation.


  


  * **Real-Time Steering:** Interrupt or redirect the conversation when details change, instead of waiting for the full response to finish.


  


  * **Stateful Conversations:** Ask AI keeps context within the active conversation, so follow-up prompts can build on earlier messages.


  


  * **Spoken Updates:** Ask AI can provide spoken summaries, confirmations, and progress updates while working through a request.


  


  * **Flexible Input:** Pause Voice Mode and continue the same conversation in text when speaking is no longer convenient.


  


  * **Safer Action Handling:** Ask AI asks for confirmation and required details before executing sensitive actions.


* * *

## **Voice Mode Availability in Labs**

  


Labs features give users early access to new HighLevel functionality while the experience is still being refined. Since Ask AI Voice Mode is available through Labs, users should expect ongoing improvements, fixes, and performance updates based on feedback.

  


To access Voice Mode, go to **Account Settings → Labs** and enable **Ask AI Voice Mode** when it is available in your account.

  


Important notes:

  


  * Voice Mode must be enabled from Labs before it can be used.


  


  * Labs features may change as HighLevel continues improving them.


  


  * Users may experience bugs, delays, or rough edges during early access.


  


  * If Voice Mode is not visible in Labs, the feature may not be available for that account yet.

## **Hands-Free Execution**

  


Hands-free execution allows users to speak requests to Ask AI and continue working without typing every instruction. This is useful when managing tasks, checking information, or working through supported actions while moving between screens or responsibilities.

  


With Voice Mode active, users can speak naturally to Ask AI. Voice Mode can help trigger supported Ask AI actions, including workflows, lead management, pipeline updates, and data retrieval where available.

  


Example voice prompts:

  


  * “Show me a summary of my recent leads.”


  


  * “Help me update this pipeline stage.”


  


  * “What are the next steps for this workflow?”


  


  * “Summarize the latest activity for this contact.”


  


  * “Create a follow-up task for this lead.”


  


Ask AI may ask for more details before continuing, especially when the request is unclear or requires a specific record, pipeline, workflow, or action.

* * *

## **Interrupt and Steer in Real Time**

  


Real-time steering helps users stay in control while Ask AI is listening or responding. If details change, or if Ask AI starts moving in the wrong direction, users can interrupt and provide updated instructions.

  


This is helpful during longer conversations or multi-step tasks where users may need to correct, narrow, or redirect the request.

  


Examples of real-time steering:

  


  * “Wait, use tomorrow instead of today.”


  


  * “Actually, only apply that to hot leads.”


  


  * “Stop there and summarize what you have so far.”


  


  * “Change the tone to be more professional.”


  


  * “Use the same idea, but make it shorter.”


  


Voice Mode is designed to adapt to the latest instruction in the conversation so users can guide the outcome as they go.

* * *

## **Stateful Conversations and Spoken Updates**

  


Stateful conversations allow Ask AI to keep track of context within the active chat. This helps users avoid repeating the same details while working through a multi-step request.

  


For example, after asking Ask AI to summarize a contact, a user might follow up with, “Now create a task for that person,” and Ask AI can use the current conversation context to understand the follow-up.

  


Voice Mode can also provide spoken updates, such as:

  


  * Confirming what it understood


  


  * Asking for missing information


  


  * Summarizing the next step


  


  * Giving progress updates during a request


  


  * Confirming before taking sensitive actions


  


These spoken updates help users stay informed without needing to watch the screen constantly.

* * *

## **Pause Voice Mode and Switch to Text**

  


Voice Mode is flexible, so users can pause speaking and continue the same conversation by typing. This helps when moving from a hands-free environment to a quiet workspace, meeting, or situation where voice input is no longer convenient.

  


Users can pause Voice Mode at any time and continue with text prompts. When switching to text, the conversation context remains available in the active Ask AI chat.

  


Use this option when:

  


  * You no longer want Ask AI to listen for voice input


  


  * You need to enter more detailed or sensitive information


  


  * You are in a meeting or shared workspace


  


  * You prefer to review or refine the next instruction before sending it


  


  * You want to stop audio responses and continue silently


* * *

## **Safe Execution for Critical Actions**

  


Safe execution helps protect users from accidental changes when Voice Mode is used for important actions. When a request involves sensitive or higher-impact changes, Ask AI asks for confirmation and any required details before continuing.

  


This helps ensure the user stays in control before Ask AI performs an action that may affect contacts, workflows, pipelines, records, or other business data.

  


Ask AI may ask for confirmation when:

  


  * The request is missing required information


  


  * The action could affect important data


  


  * The request involves changing records or statuses


  


  * The scope of the action needs clarification


  


  * The action requires user approval before execution


  


Users can approve, cancel, or revise the request before Ask AI continues.

## **How To Setup Ask AI Voice Mode**

  


Proper setup ensures Voice Mode is enabled and ready for use before users begin speaking to Ask AI. Since Voice Mode is available through Labs, users must enable it first and then open Ask AI to start a voice session.

  


  1. In HighLevel, go to **Account Settings**.
  2. Click **Labs**.
  3. Find **Ask AI Voice Mode**.
  4. Toggle the feature **On**.
  5. Open **Ask AI** from the available Ask AI entry point in HighLevel.
  6. Locate the Voice Mode or microphone control in the Ask AI workspace.
  7. Click the Voice Mode or microphone control to begin.
  8. Allow microphone access if prompted.
  9. Speak your request naturally.
  10. Listen for Ask AI’s response or review the response in the chat.
  11. Interrupt, pause, or switch to text whenever needed.
  12. Confirm or revise any sensitive action before Ask AI executes it.

## **Frequently Asked Questions**

  


**Q: What is Ask AI Voice Mode?**  
Ask AI Voice Mode lets users speak to Ask AI instead of typing. Users can ask questions, receive spoken responses, trigger supported actions, interrupt the conversation, pause voice, and switch back to text.

  


**Q: How do I enable Ask AI Voice Mode?**  
Go to **Account Settings → Labs** , find **Ask AI Voice Mode** , and toggle it on when it is available in your account.

  


**Q: Why do I not see Ask AI Voice Mode in Labs?**  
Voice Mode may not be available for your account yet, or you may not have the required access to enable Labs features. Check with your admin if the option is missing.

  


**Q: Can I switch from voice to text?**  
Yes. You can pause Voice Mode and continue the same Ask AI conversation by typing in the composer.

  


**Q: Can I interrupt Ask AI while it is responding?**

Yes. Voice Mode supports real-time steering, so you can interrupt, correct details, or redirect the conversation while working through a request.

  


**Q: Will Ask AI ask before performing sensitive actions?**  
Yes. Ask AI asks for confirmation and any required details before executing sensitive actions.

  


**Q: Does Voice Mode remember the conversation context?**  
Voice Mode keeps context within the active conversation, so follow-up prompts can build on earlier messages in the same chat.

  


**Q: Is Ask AI Voice Mode the same as Voice AI Agent or Voice AI Chat Widget?**  
No. Ask AI Voice Mode is for HighLevel users speaking to Ask AI inside the platform. Voice AI Agents and Voice AI Chat Widget are separate features used for AI-powered voice or visitor conversations.

  


**Q: What should I do if Voice Mode has bugs or delays?**  
Since Voice Mode is available through Labs, feedback is important. Share what you asked, what happened, what you expected, and any delays or unexpected behavior you noticed.
