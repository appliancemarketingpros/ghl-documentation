# Workflow Action - Go To

**Source URL:** [https://help.gohighlevel.com/support/solutions/articles/48001196760-workflow-action-go-to](https://help.gohighlevel.com/support/solutions/articles/48001196760-workflow-action-go-to)  
**Category:** Workflows  
**Folder:** Internal Tools Workflow Actions

---

The Go-To event allows you to jump contacts from one event to another in Workflows. To use a Go-To step, simply add a Go-To event and then draw a line to where you want the contacts to jump - that's it!
