# **What are Custom Fields?**  
  


Custom fields allow you to store additional information about contacts or opportunities beyond the default fields already available in your accounts.  
  


These fields can be used across:  
  


  * Forms  
  

  * Surveys  
  

  * Pipelines  
  

  * Workflows  
  

  * Automations  
  

  * Smart Lists  
  

  * Contact records  
  

  * Opportunity records  
  


Custom fields make it possible to collect business-specific details such as:

  * Lead type  
  

  * Budget range  
  

  * Preferred contact method  
  

  * Referral source  
  

  * Property type  
  

  * Timeline to close  
  


This helps create cleaner records, better segmentation, and more personalized automation.

* * *

## **Key Benefits of Custom Fields**

  
Custom fields provide flexibility and structure across your accounts and automation setup.

Benefits include:  
  


  * Collect business-specific data  
  

  * Organize contact and deal information more efficiently  
  

  * Personalize email and SMS automation using merge fields  
  

  * Build better Smart Lists and workflow filters  
  

  * Display important information directly inside records and pipelines  
  

  * Group related fields together using folders for cleaner layouts


  


* * *

## **Contact vs. Opportunity Custom Fields**  
  


Choosing the correct field type is important because contact fields and opportunity fields behave differently inside the system.  
  


### **Contact Custom Fields**  
  


Contact custom fields are attached directly to a person.  
  


Use contact fields for information such as:  
  


  * Buyer or seller type  
  

  * Birthday  
  

  * Preferred contact method  
  

  * Referral source  
  

  * Business website  
  


These fields can be used inside:  
  


  * Contact records  
  

  * Forms  
  

  * Surveys  
  

  * Contact workflows  
  

  * Smart Lists  
  

  * Email and SMS personalization  
  


### **Opportunity Custom Fields**  
  


Opportunity custom fields belong to a specific deal or sales opportunity.

Use opportunity fields for information such as:  
  


  * Budget range  
  

  * Property type  
  

  * Deal urgency  
  

  * Timeline to close  
  

  * Sales stage details  
  


These fields appear inside:  
  


  * Opportunity records  
  

  * Pipeline views  
  

  * Opportunity workflows  
  

  * Deal reporting and automation


  

    
    
    **Note:** Once a field is created as either a contact or opportunity field, it cannot be switched.

## **Organizing Custom Fields with Folders**  
  


Folders help organize related fields into sections so records remain clean and easy to navigate.  
  


Folders appear directly inside record layouts and can group related information together.  
  


Examples:  
  


  * Buyer Information  
  

  * Property Details  
  

  * Sales Qualification  
  

  * Appointment Preferences  
  


### **Creating a Folder**  
  


To create a folder:  
  


  * Open Settings  
  

  * Navigate to Custom Fields  
  

  * Choose the object using the object selector  
  

  * Open the Folders tab  
  

  * Click Create Folder  
  

  * Enter the folder name  
  

  * Save the folder  
  


After creating folders, you can assign fields to them for better organization.

## **Frequently Asked Questions**

  


**Q: Can I convert a contact field into an opportunity field (or vice versa)?**  
No. You must recreate the field under the correct object type. Field objects cannot be changed after creation.

  


**Q: What types of fields can I create?**  
You can create Short Text, Long Text, Drop downs, Date Pickers, Phone, Email, Radio Selects, Checkbox Groups, URLs, and more.

  


**Q: Why don’t my fields show in a form or workflow?**  
Ensure the object type matches the use case. Only Contact fields appear in forms and contact workflows.

  


**Q: How many options can the Dropdown (multiple) custom field have?**

The Dropdown (multiple) field can have a lot of options (50+) so there is no practical limit.
