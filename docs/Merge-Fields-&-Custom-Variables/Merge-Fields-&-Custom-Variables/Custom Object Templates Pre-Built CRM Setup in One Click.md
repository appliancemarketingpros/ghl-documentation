# **What are Custom Object Templates?**

  


Custom Object Templates are pre-built configurations that help you quickly create Custom Objects for common industries and use cases. Instead of manually deciding which objects, fields, folders, and associations to create, you can start with a ready-made structure and adjust it to fit your needs.

  


Custom Objects are used to track business data that does not fit neatly into standard CRM records like Contacts, Companies, or Opportunities. Templates make that setup faster by giving you a practical starting point based on common workflows.

  


Custom Object Templates are available in the Template Library alongside other template modules. You can browse supported industries, preview template details, and apply a template when creating Custom Objects.

* * *

## **Key Benefits of Custom Object Templates**

  


Custom Object Templates help teams launch structured CRM setups faster while still allowing flexibility for your processes.

  


  * **Faster Setup:** Create Custom Objects, fields, folders, and associations in one flow instead of building each part manually.  
  

  * **Industry-Specific Starting Points:** Choose from templates designed for common industries such as Real Estate, Insurance, Automotive, SaaS, Events, Pets, and Home Services.  
  

  * **Easier Data Organization:** Use predefined fields and folders to keep related information grouped clearly.  
  

  * **Built-In Relationship Mapping:** Templates can include associations so records can be connected across Companies, Opportunities, and Custom Objects.  
  

  * **Preview Before Applying:** Review the template’s objects, fields, associations, description, tags, and prerequisites before using it.

## **Template Library Visibility Controls**

Applying a Custom Object Template creates a ready-made object structure you can use as a starting point for your CRM setup. Reviewing the template before applying it helps confirm the objects, fields, and associations match your process.

## **Frequently Asked Questions**

  


**Q: Do Custom Object Templates create records automatically?**

A: No. Custom Object Templates are designed to create the CRM structure (Custom Objects, fields, folders, and associations). They do not create customer records or sample records.

  
**Q: Can I edit a template after applying it?**

A: You can edit the Custom Objects that were created by the template. After importing, review the created objects, fields, and associations and adjust them as needed.

  
**Q: Will applying a Custom Object Template overwrite my existing Custom Objects?**

A: Templates are designed to add a new structure. Review the template preview before importing and verify your setup after importing.

  
**Q: What happens if a template is hidden after it has already been used?**

A: Assets created from templates generally remain available for editing even if browsing or importing new templates is later restricted.

  
**Q: Do Custom Object Templates include associations?**

A: Yes. Templates can include associations to help connect related records such as Companies, Opportunities, and Custom Objects.

  
**Q: Are workflows or sample data included with Custom Object Templates?**

A: Not at this time. Templates currently focus on object structure (objects, fields, folders, and associations).

  
**Q: Can users create and save their own Custom Object Templates?**

A: Not at this time.

  
**Q: Which industries are currently supported?**

A: Current templates include Real Estate, Insurance, Automotive, SaaS, Events, Pets, and Home Services.
